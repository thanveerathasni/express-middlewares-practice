# Audit Log Middleware

## What is audit logging?
Audit logging records important user actions
for security, compliance, and tracking.

## Why use it?
- Track sensitive operations
- Security audits
- Compliance requirements

## How it works?
- Middleware logs user + action
- Stores timestamp and route info

## Where it is used?
- Fintech applications
- Admin panels
- Enterprise systems

## Metrics vs Audit
- Metrics: performance data
- Audit: user action history

## Common mistakes
- Logging too much data
- Logging sensitive information

## Interview one-liner
Audit middleware records critical
user actions for accountability.
